# 产品需求文档

> 版本：v0.1（2026-05-14）
> 状态：MVP 范围已确认

## 1. 产品定位

一款**跨平台的本地小说阅读器**，主要面向移动端使用，桌面端为辅。专注于把"导入 txt 小说 → 看小说 → 跨书全文搜索"这条主路径做扎实，不做联网书源、不做云同步、不做社区。

## 2. 目标用户与场景

- 个人使用者，习惯把 txt 小说放在本地（云盘下载、群文件等）
- 主要在 Android 手机上阅读，偶尔在 Mac/Windows/Linux 桌面上看
- 单设备使用，不需要跨设备同步进度
- 需要在自己的整个书库里搜内容（"上次看到那个 XX 角色出现在哪本书来着？"）

## 3. 平台支持

| 平台 | 优先级 | 说明 |
|---|---|---|
| Android | P0（主战场） | 手机阅读是核心场景 |
| macOS | P1 | 桌面端使用 |
| Windows | P1 | 桌面端使用 |
| Linux | P1 | 桌面端使用 |
| iOS | 暂不考虑 | 无设备、无开发者账号 |
| Web | 暂不考虑 | 与本地文件场景不匹配 |

## 4. 功能需求（MVP）

### 4.1 文件导入

- 支持从系统选择一个或多个 txt 文件导入
- 导入时**拷贝到 app 沙盒目录**（避免原文件被移动/删除导致失效）
- 自动检测文本编码：BOM 检查 → UTF-8 严格解码 → 失败则 fallback 到 GBK/GB18030
- 导入流程异步执行，UI 显示进度（"正在解析章节"、"正在建立索引"）
- 默认书名取自文件名（去扩展名），用户可后续修改

### 4.2 章节解析

- 自动识别中文小说常见章节标题模式，生成可跳转目录：
  - `第X章` / `第X节` / `第X回` / `第X部` / `第X卷`（X 支持中文数字与阿拉伯数字）
  - `序章` / `序言` / `楔子` / `引子` / `尾声` / `后记` / `前言`
  - `Chapter X`（兼容英文）
- 当一本书识别不到任何章节时，将整本书作为单章节处理（不报错）

### 4.3 阅读器

- **分页模式**，无翻页动画（点击切换，响应快）
- 翻页交互：屏幕左/右点击翻页，中间唤起菜单
- 自动记录每本书的阅读进度，下次打开自动恢复
- 支持从目录跳转到指定章节
- 支持上一章 / 下一章快捷跳转

### 4.4 跨书全文搜索

- 全局搜索框：输入关键词，跨书库所有书检索
- 结果列表展示：书名 + 章节标题 + 关键词上下文片段
- 点击搜索结果跳转到对应位置（书 + 章节 + 段落）
- 索引在导入时异步建立，搜索时直接查询，不重新扫描全文

### 4.5 书架

- 网格 / 列表两种视图切换
- 显示书名、最近阅读时间、阅读进度百分比
- 支持删除书籍（同时清除文件、元数据、索引）
- 按书名 / 作者过滤（不是全文）

### 4.6 设置

- 字号大小（小 / 中 / 大 / 超大，4 档）
- 行距（紧凑 / 标准 / 宽松，3 档）
- 主题（白底黑字 / 黑底白字 / 米黄护眼，3 套）
- 字体：仅使用系统字体（不内置字体，不支持用户导入 ttf——MVP 范围）

## 5. 非功能需求

### 5.1 性能

- 单本 txt 文件目标支持范围：1-10 MB（中长篇小说）
- 导入一本 5 MB 的小说，从选择文件到可阅读 ≤ 10 秒（含编码检测、章节解析、索引建立）
- 翻页响应 ≤ 100 ms（无动画，应感知不到延迟）
- 全库搜索（书库 ≤ 100 本）响应 ≤ 1 秒

### 5.2 兼容性

- Android API Level 26+（覆盖 95%+ 在用机型）
- macOS 10.15+
- Windows 10+
- Linux：以主流发行版（Ubuntu 20.04+、Debian 11+、Fedora 36+）为目标，不保证小众发行版

### 5.3 可靠性

- App 崩溃时，下次启动能恢复到上次阅读的书与章节
- 文件导入失败时，给出明确错误信息（编码不识别、文件损坏等），不残留半成品数据

### 5.4 隐私

- 完全本地运行，无任何网络请求
- 不收集用户数据、不上报崩溃日志（MVP 不接入任何 SDK）

## 6. 明确不做的事（MVP 范围外）

- 联网书源 / 网络小说搜索
- 多端云同步阅读进度与书架
- 社交功能（评论、书评、分享）
- 听书 / TTS 朗读
- 翻页仿真动画（书页翻折效果）
- 内置字体或用户字体导入
- 多种文件格式（仅 txt，epub 在 v0.2 路线图）

## 7. 路线图（已识别但延后）

| 版本 | 主题 | 内容 |
